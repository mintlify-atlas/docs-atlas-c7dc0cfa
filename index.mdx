Gas Town lets you coordinate 20–30 AI coding agents (Claude Code, GitHub Copilot, Codex, Gemini, and others) working in parallel across your projects. Work state persists in git-backed hooks — agents survive restarts, crashes, and context resets without losing progress.

## What Gas Town solves

| Challenge | Gas Town solution |
|-----------|------------------|
| Agents lose context on restart | Work persists in git-backed hooks |
| Manual agent coordination | Built-in mailboxes, identities, and handoffs |
| 4–10 agents become chaotic | Scale comfortably to 20–30 agents |
| Work state lost in agent memory | Work state stored in Beads ledger |

## Get started

<Steps>
  <Step title="Install Gas Town">
    ```bash
    brew install gastown
    # or: go install github.com/steveyegge/gastown/cmd/gt@latest
    ```
  </Step>
  <Step title="Create your workspace">
    ```bash
    gt install ~/gt --git
    cd ~/gt
    ```
  </Step>
  <Step title="Add a project">
    ```bash
    gt rig add myproject https://github.com/you/repo.git
    ```
  </Step>
  <Step title="Start the Mayor">
    ```bash
    gt mayor attach
    ```
    Tell the Mayor what you want to build — it handles the rest.
  </Step>
</Steps>
